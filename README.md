4D Plugin Wizard (miyako version)
====================

Based on new SDK.

Added Date Array, Time Array classes.

BLOB array, Picture array are not mapped to classes.

OBJECT, OBJECT Array, BOOLEAN, not supported.

Configured for 4D Cocoa.
